  }
  What it does:
  - rooms.get(roomId) - Try to find the room in our Map
  - If it doesn't exist (!room), create a new Room object
  - rooms.set(roomId, room) - Store the new room in our Map

  Example: First person joins "team-alpha" → creates the room. Second person joins "team-alpha" → uses existing room.

  ---
  3. Create Player Object

  const player = {
    id: socket.id,
    name: playerName,
    socketId: socket.id,
  }
  What it does: Creates a player object with their info.
  - socket.id - Unique ID like "abc123" assigned by Socket.io

  ---
  4. Add Player to Room

  room.players.set(socket.id, player)
  What it does: Adds the player to the room's players Map.
  - Key = socket.id (so we can find them later)
  - Value = player object

  ---
  5. Join Socket to Room

  socket.join(roomId)
  What it does: This is a Socket.io feature that groups sockets into "rooms".

  Why? So we can broadcast messages to everyone in that room:
  io.to('team-alpha').emit('message', data)  // Only people in team-alpha get this

  ---
  6. Broadcast Room Update

  io.to(roomId).emit('room-update', {
    players: Array.from(room.players.values()),
    revealed: room.revealed,
  })
  What it does:
  - io.to(roomId) - Send to everyone in this room
  - .emit('room-update', {...}) - Send event called "room-update" with data
  - Array.from(room.players.values()) - Convert the Map to an array

  Why send this? All clients in the room need to know:
  - Who's in the room (players array)
  - Are votes revealed? (revealed boolean)
